# Personal Portfolio Website

  A responsive personal portfolio website built using React, HTML, CSS, and JavaScript.
  This portfolio showcases my projects, certifications, achievements, and extracurricular activities.

## Features
- Responsive design for desktop and mobile devices
- Dark and light theme toggle
- Interactive project cards
- Image modal preview system
- Mobile navigation menu
- Smooth page transition animations
- Certifications and achievements section
- Contact section with social links
## Technologies Used
- React.js
- HTML5
- CSS3
